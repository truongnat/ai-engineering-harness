# Core Loop

The core loop is the default operating sequence for this harness.

1. Start by reading artifacts and mapping the workspace.
2. Discuss the goal until scope and constraints are explicit.
3. Write a plan before making changes.
4. Execute the plan in small steps.
5. Verify with fresh evidence.
6. Ship with a clear summary.
7. Record durable memory.

Use this workflow when no more specific workflow is a better fit.
