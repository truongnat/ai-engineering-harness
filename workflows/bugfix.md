# Bugfix Workflow

Use this workflow when restoring expected behavior.

1. Start and map the failing area.
2. Clarify the bug, reproduction, and expected behavior.
3. Write a plan including regression protection.
4. Execute the fix with the smallest practical change.
5. Verify the reproduction no longer fails and that related behavior still works.
6. Ship with a brief root-cause summary.
7. Remember any durable lesson or hazard.
