# Command Loop

The command loop gives agents a predictable sequence:

1. `harness-start`
2. `harness-map`
3. `harness-discuss`
4. `harness-plan`
5. `harness-run`
6. `harness-verify`
7. `harness-ship`
8. `harness-remember`

Not every task requires every command, but the loop defines the default progression and prevents coding before planning.
